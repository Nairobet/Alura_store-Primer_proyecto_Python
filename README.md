# Alura_store-Primer_proyecto_Python
Este proyecto realiza un análisis integral del desempeño de cuatro tiendas, con el fin de identificar cuál presenta el menor aporte en facturación, tracción comercial y potencial de crecimiento. Incluye exploración de datos, generación de indicadores, visualizaciones estratégicas y una recomendación ejecutiva final basada en evidencia.
